# 代码编辑器（Neovim）
因为 Arch Linux 上直接使用 pacman 安装的 vim 是-clipboard，无法使用系统剪贴板。解决办法就是用 neovim 顶掉 vim ，不过不用担心，安装了 neovim 后还是可以在命令行里使用 vim 的，只不过这个 vim 就有 +clipboard 了。
```
$ sudo pacman -S neovim
```
